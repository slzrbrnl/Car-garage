# Car garage manager
A command-line application to manage a car garage: add, search, update, delete and track average horsepower. Data persist between sessions using JSON.

## Features

- Create your personalized garage
- Add the cars you want
- Show your saved cars with one command
- Find your added cars
- Get your average HP
- Eliminate cars you no longer want
- Update your cars
- Save your garage automatically
- Load your garage on startup

## How to run

```bash
python3 garage.py
```

You'll see a menu with options:

1. Add your car by entering brand, model, year, and horsepower
2. Show your garage with the cars you have already saved
3. Search for a car by brand
4. Eliminate a car by entering the brand
5. Leave the program

## Tech stack

- Python 3
- JSON for data persistence
